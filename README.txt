This application is a backend service created using Nodejs and MongoDB.
The major modules of NodeJS utilised- npm, express
The routing is done using RESTAPI and provides facility to add, view, delete, modify and update database content.
It also two levels of authentication for deleting data and other activites.
Admin access only can delete the data and User access only can add, modify, view and update data.
So it provides a idea of login to users and utilises JSON Web token for admin access.
